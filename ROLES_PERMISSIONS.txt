IZOGRUP CRM - ROLES & PERMISSIONS CONFIGURATION
================================================

PREDEFINED ROLES (9 Total):
1. Admin - Full system access
2. Product Manager - Manage products and services
3. Site Manager - Manage construction sites and workers
4. Offer Manager - Create and manage offers
5. Order Manager - Manage orders and sales agents
6. Sales Agent - Create orders and manage clients
7. Office Employee - Process orders
8. Worker - View assigned tasks
9. Website Manager - Manage website content

PERMISSION TYPES:
================
- Read: View resource
- Create: Add new items
- Update: Edit existing items
- Delete: Remove items
- Export: Download/export data
- Approve: Approve actions/changes

NOTES:
======
- All roles are system roles (cannot be deleted)
- Admin can add custom roles through UI
- Permissions are enforced at API level
- UI elements are hidden based on permissions
- Chat is available for all roles (pending implementation)
- Reports module available for managers and admin
